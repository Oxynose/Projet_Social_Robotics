## Test de l'agent PPO

Pour entraîner/tester l'agent ppo:
```bash
python main.py --ppo --train
python main.py --ppo --play
```

Pour entraîner/tester l'agent gail:
```bash
python main.py --gail --train
python main.py --gail --play
```

Sinon pour juste lancer RaicingCar :
```bash
python main.py
```
